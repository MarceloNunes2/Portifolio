Sobre Mim

Sou um desenvolvedor com formação em Ciência da Computação e experiência em desenvolvimento front-end utilizando HTML, CSS e JavaScript. Além disso, tenho conhecimentos básicos em PostgreSQL e C#. Atualmente, estou estudando e me aprofundando em tecnologias como React e Node.js, o que me permitirá desenvolver soluções mais robustas e escaláveis. Já estagiei como desenvolvedor C#, o que me deu a oportunidade de trabalhar em projetos reais e adquirir habilidades de programação e resolução de problemas. Estou sempre em busca de novos desafios e aprendizados, e estou animado em aplicar minhas habilidades para criar soluções criativas e eficientes para os problemas de meus clientes

Projeto 1
Site de Beleza Design Joyce

Desenvolvi um site de beleza totalmente responsivo para uma cliente que oferece serviços de cuidados com a pele, maquiagem e penteados. O site foi construído utilizando HTML, CSS e JavaScript para garantir uma experiência de usuário agradável e interativa em qualquer dispositivo, desde desktops até smartphones. O site apresenta um design elegante e sofisticado que reflete a marca e os serviços oferecidos pela cliente. Utilizei imagens de alta qualidade para destacar os tratamentos disponíveis e incluí uma seção de depoimentos para demonstrar a satisfação dos clientes anteriores. Também implementei um sistema de reservas online para facilitar o agendamento você pode enviar um e-mail assim pode agendar seu horario. Também tem a opção de chamar via Whatssap com um botão flutuante que deixei na página Ao desenvolver este site, foquei na usabilidade e na acessibilidade, garantindo que ele fosse fácil de navegar e utilizável em diversos dispositivos e navegadores. Estou orgulhoso do resultado final e acredito que este site irá ajudar minha cliente a expandir seus negócios e a alcançar mais clientes em potencial.

Projeto 2
Conversor de Moedas

Este é um projeto de conversor de moedas em tempo real desenvolvido usando HTML, CSS e JS. Ele permite aos usuários converter valores de uma moeda para outra com base nas taxas de câmbio mais recentes, fornecidas por uma API externa. A interface do usuário é simples e intuitiva, permitindo que os usuários selecionem as moedas de origem e destino, insiram o valor a ser convertido e obtenham instantaneamente o resultado da conversão. Para desenvolver este projeto, utilizei habilidades em HTML para definir a estrutura da página, CSS para estilizar a interface do usuário e torná-la responsiva em diferentes dispositivos, e JS para gerenciar a lógica de conversão de moedas, validação de entrada de usuário e chamadas AJAX para a API externa. Este projeto demonstra minhas habilidades em desenvolvimento web e minha capacidade de criar aplicativos úteis e práticos para usuários em diferentes áreas. Se você precisar de um desenvolvedor de web para criar um projeto semelhante ou outra aplicação web, sinta-se à vontade para entrar em contato comigo.

Projeto 3
Clima/Tempo

Este é um projeto de clima mundial em tempo real, desenvolvido utilizando HTML, CSS e JS. O objetivo deste projeto é permitir aos usuários obter informações precisas e atualizadas sobre as condições climáticas em diferentes locais ao redor do mundo, com base em dados fornecidos por uma API externa. A interface do usuário deste projeto é simples e intuitiva, permitindo que os usuários pesquisem cidades específicas ou usem a geolocalização para obter informações climáticas de sua localização atual. O aplicativo exibe informações detalhadas sobre a temperatura atual, previsão do tempo, umidade e outros dados relevantes, em tempo real. Para desenvolver este projeto, utilizei habilidades em HTML para definir a estrutura da página, CSS para estilizar a interface do usuário e torná-la responsiva em diferentes dispositivos, e JS para gerenciar a lógica de busca de dados climáticos, validação de entrada de usuário e chamadas AJAX para a API externa.

Projeto 4
Cards-Pokémon

A interface do usuário deste projeto é baseada em cards de Pokémon, permitindo que os usuários naveguem pelas diferentes cartas exibidas na tela e cliquem em um card para obter informações detalhadas sobre o Pokémon em questão. Além disso, o projeto inclui um mecanismo de pesquisa que permite aos usuários procurar cartas específicas de Pokémon com base em seu nome ou tipo. Para desenvolver este projeto, utilizei habilidades em HTML para definir a estrutura da página, CSS para estilizar os cards de Pokémon e torná-los atraentes visualmente, e JS para gerenciar a lógica de exibição e busca de cartas de Pokémon, validação de entrada de usuário e a inserção manual dos dados dos cards de Pokémon.
